# Calla's Coffee - Order Management System

Sistem manajemen pesanan kopi kekinian, estetik, dan premium.

## ☕ Tentang Proyek
Sistem ini memfasilitasi pelanggan untuk memesan kopi secara mandiri melalui landing page yang didesain secara elegan dengan nuansa kafe premium, serta menyediakan dashboard admin bagi pengelola kafe untuk memantau dan mengelola pesanan.

## 🚀 Fitur Utama
- **Landing Page User (Frontend)**: Desain estetik dan premium menggunakan HTML murni, Tailwind CSS, dan Vanilla JavaScript. Memiliki fitur keranjang belanja dinamis dan perhitungan total harga.
- **Admin Dashboard (Backend - Dalam Pengembangan)**: Berbasis Laravel 11 untuk kelola menu, pesanan, dan status pembayaran.

## 🛠️ Cara Instalasi
1. Pastikan Anda memiliki PHP 8.2+ dan Composer terinstal.
2. Buka direktori proyek ini (`c:\laragon\www\Andra`).
3. Jalankan `composer install`
4. Salin `.env.example` ke `.env` dan konfigurasikan database Anda.
5. Jalankan `php artisan key:generate`
6. Jalankan `php artisan migrate --seed`
7. Jalankan `php artisan serve` untuk mengakses aplikasi lokal.
